### 개요

이 프로그램은 서버로부터 받은 파일과 셀 ID에 대한 정보를 바탕으로 각 셀의 배경색을 추출하여, 배경색을 기준으로 포지션을 분류하고, 셀ID와 포지션을 합쳐 서버로 다시 보내는 fastapi 서버이다. 해당 서버에서는 셀의 배경색을 추출하는 데 특화된 기술인 OpenCV를 이용하여, 배경색을 추출하는 부분에 집중된 서비스이다.

### 기능

- Spring boot 서버로부터 도착한 사진과 request를 객체로 받는다.
- OpenCV 기술을 이용하여 배경색을 추출한다.
- 배경색을 일정 기준에 맞도록, 분류하고 이를 셀ID와 합친다.
- 셀ID와 배경색이 담긴 정보를 Spring boot로 재전송한다.
- OpenCV는 BGR 순서로 추출하며, B의 비율이 R보다 일정 수준 높은 경우, ‘웰컴’ 포지션으로 분류한다.
- R의 비율이 B보다 일정 수준 높은 경우, ‘엔젤’ 포지션으로 분류한다.
- 세 색깔의 비율이 비슷한 경우, ‘매점’ 포지션으로 분류한다.

### 요청 예시

```jsx
{
  "image_width": 1200,
  "image_height": 800,
  "cells": [
    {
      "cell_id": 371,
      "normalized_vertices": [
        [0.7811683, 0.078998074],
        [0.8230166, 0.078998074]
      ]
    }
  ]
}
```
